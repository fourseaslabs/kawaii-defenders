# Kawaii Defenders Academy 🐰🐻🚓🐼🦉

An educational tower defense game that makes learning fun for kids and adults.

## 🎮 Play Now

**[kawaii-defenders.vercel.app](https://kawaii-defenders.vercel.app)**

## Features

### 🧒 Kids Mode (Singapore MOE Syllabus)
- **Levels:** K2, P1, P2, P3
- **Subjects:** Math, English, 华文 (Chinese), Mixed
- **Curriculum:** Aligned to My Pals Are Here! Maths, STELLAR, 欢乐伙伴 (Huan Le Huo Ban)
- Gentle pacing with 5-7 lives, more starting coins

### 🧑 Adults Mode (Language Learning)
- **Korean 🇰🇷** — Beginner (TOPIK I) with 280+ questions
- Staged curriculum across 500 levels: Hangul → Greetings → Numbers → Vocabulary → Verbs → Particles → Sentences → Real Life → Grammar
- More coming: Intermediate (TOPIK II), Advanced, Native tiers
- Japanese 🇯🇵 planned

### 🏰 Tower Defense Gameplay
- 8 unique towers (🐰🐻🚓🚗👸🐼🦄🦉), each upgradable to Level 10
- 8 rounds per level on the same map — build up your defenses!
- Boss battles every 10 levels — build your own maze!
- 5 map variants, 500 levels per subject, procedural difficulty scaling

### 📚 Learning System
- **Study cards** before each quiz — learn before being tested
- **Spaced repetition** — wrong answers come back for review
- **Hint button** on review questions
- **Explanations** on every question (413 total)
- Questions never repeat unless you got them wrong

## Tech

Single HTML file. No build step. No dependencies.
- Canvas-based tower defense with emoji characters
- Web Audio API synthesized music & sound effects
- localStorage for progress persistence
- BFS pathfinding for boss maze validation
